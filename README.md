# Customer_behavior_Dashboard
Data Analyst project showcasing customer behavior using python, SQL, Power BI

 Project Overview

This project focuses on performing an end-to-end data analytics workflow using Python, SQL, and Power BI. The main objective was to extract insights from the dataset through data cleaning, exploratory data analysis (EDA), SQL querying, and interactive dashboard creation.

The project demonstrates practical skills in:

Data loading and preprocessing
Exploratory Data Analysis (EDA)
SQL database querying
Data visualization
Business reporting and presentation
